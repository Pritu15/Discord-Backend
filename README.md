# Discord Backend

## Requirements

- Docker
- Docker Compose

## Setup

1 Clone repository

git clone https://github.com/Pritu15/Discord-Backend.git
2 Go to project

cd discord-backend

3 Create env file

cp .env.example .env

4 Add database url inside .env

5 Run docker

docker compose up --build

Server will start at:

http://localhost:3000
